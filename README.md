The Eden Bot's source code can be found here. If you would like access to the developer channels in the Discord, please make a ticket.

### AI

We allow the usage of AI generated code so long as you, the author of the PR, are able to understand and explain the intent and effect of the solution your AI creates. We do not want to waste time talking directly to your AI.
